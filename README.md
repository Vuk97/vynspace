# VYN Space

Futuristic product website for VYN Space, a digital onboarding platform helping refugees, students, and skilled workers integrate into Germany through housing, finance, jobs, and campus services.

## Tech

- React
- Vite
- Lucide React icons
- GitHub Pages workflow

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

## Deployment

The repository includes `.github/workflows/deploy.yml` for GitHub Pages. After pushing to `main`, enable Pages with **GitHub Actions** as the source in repository settings.
